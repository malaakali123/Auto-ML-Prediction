## âš™ï¸ 2. How It Works (Kaam Kese Karta Hai?)
System 4 simple steps mein kaam karta hai:

1.  **Upload Data:** User apna CSV file (e.g., Sales Data, Patient Records) upload karta hai.
2.  **Auto Processing (Magic Box):**
    *   System khud **missing values** ko fill karta hai.
    *   **Text data** ko numbers mein convert karta hai.
    *   **Problem detect** karta hai (Classification hai ya Regression).
3.  **Auto Training:**
    *   System automatically **5+ powerful models** (Random Forest, XGBoost, etc.) train karta hai.
    *   Sabse best parameters dhundne ke liye **Hyperparameter Tuning** karta hai.
4.  **Results & Leaderboard:**
    *   User ko ek **Leaderboard** dikhata hai ki kaunsa model sabse acha hai.
    *   **Graphs** show karta hai taake data asaani se samajh aaye.
